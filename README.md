# Football Photography

## Author

Peace Keter

## Description

Football Photography is a personal photography portfolio website created using HTML and CSS. The website showcases my passion for football photography through a collection of favorite photos and information about my interests.

The website contains:

* About Me section
* My Interests section
* My Favorite Photos gallery
* Contact Me section
* Responsive and attractive design using CSS

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub
* GitHub Pages

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/peaceketer55-ship-it/football-photography.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
